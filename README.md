# MECH544 – Coursework II

This repository contains all MATLAB, Simulink and Simscape Multibody files used for **MECH544 Coursework II**, including:

- **Activity 1** – System identification and modelling  
- **Activity 2** – 6-DoF motion platform and maze simulation 
- **Activity 3** – Nonlinear vehicle cruise–control model with PI controller  

It also includes the original figures used in the report and scripts used for data processing and plotting.
